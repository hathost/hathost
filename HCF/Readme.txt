READ ME:
*********

Features:
=========
a) Simple way to understand the user needs.

b) Avoid spam emails by hidding your email address.

c) Easy to integrate into your website.

d) Customizable form field titles and CSS style.


Requirements:
=============
Linux or Windows server (it *should* work on any server with PHP support) 
PHP 4.3.x or high.

Usage:
------
1) Download and Unzip HCF.zip to extract the files and folders.

2) This file contains index.php,contac.php,style.css and Readme.txt files.


How to embed the script in to your webpage?
-------------------------------------------
1)Run index.php file in your browser.

2) The script code is available inside the <body> of the index.php file. 

3) In the <head> tag include 
		<link rel="stylesheet" type="text/css" href="style.css">

4) mail functions are done in the contac.php file.
